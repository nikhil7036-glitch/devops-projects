       Common Terraform Commands
        Command	      Purpose                                                                    
terraform init	 -   Initializes Terraform (downloads plugins)
terraform fmt    - 	Formats code
terraform validate	-   Checks for syntax/config errors
terraform plan	 -  Previews what will be created
terraform apply	 -  Creates/updates infrastructure
terraform apply-auto-approve	-  Applies changes without asking “yes/no”
terraform destroy	-  Deletes infrastructure
